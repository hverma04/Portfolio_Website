The Company

Quant strategy & FP&A for the underserved

  • Services span options strategy, volatility modeling, and financial planning & analysis — delivered without the investment bank price tag.
MyQuant

Live options probability model

  • Built MyQuant: a live stock options probability model using Geometric Brownian Motion (GBM) for stock price path simulation.
  • Runs 10,000 Monte Carlo iterations per query to generate probability distributions across strike prices and expiry dates.
  • Outputs expected value, win probability, and tail-risk metrics — all from a Python backend connected to live market data.
Technical Stack

Built, not bought

MyQuant is written in Python (pandas, NumPy, SciPy) with GBM-based price path simulation at its core. The 10,000-iteration Monte Carlo approach ensures statistically robust probability estimates even for low-probability tail outcomes. It is live, not a back-tested prototype.
